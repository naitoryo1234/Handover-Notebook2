# 設定ページ 動作確認台本

## アクセス方法
1. `http://localhost:3000` を開く
2. ヘッダー右上の **歯車アイコン** をクリック → `/settings` に遷移

---

## テスト1: スタッフ管理

### 1-1. スタッフ一覧確認
- [ ] 「スタッフ管理」タブが表示されている
- [ ] スタッフ一覧が表示される（田中、鈴木、院長など）
- [ ] 各スタッフに編集ボタン（鉛筆）、有効/無効トグルがある

### 1-2. スタッフ追加
1. 「**+ 追加**」ボタンをクリック
2. 名前: `テスト太郎`、役職: `施術者` を入力
3. 「追加」ボタンをクリック
4. → 一覧に「テスト太郎」が追加されることを確認

### 1-3. スタッフ編集
1. 任意のスタッフの **鉛筆アイコン** をクリック
2. 名前や役職を変更
3. 「更新」ボタンをクリック
4. → 変更が反映されることを確認

### 1-4. 有効/無効切り替え
1. 任意のスタッフの **トグルアイコン** をクリック
2. → 無効になると薄くなることを確認
3. 再度クリックで有効に戻ることを確認

---

## テスト2: AIプリセット

### 2-1. プリセット一覧確認
1. 「**システム設定**」タブをクリック
2. → 「AIプリセット」セクションが表示される
3. → 「デフォルト」と「鍼灸院」の2つのオプションがある

### 2-2. プリセット切り替え
1. 「**デフォルト**」をクリック
2. → 「保存しました」メッセージが表示される
3. → デフォルトにチェックマークがつく
4. 「**鍼灸院**」をクリックして戻す

### 2-3. 永続化確認
1. プリセットを「デフォルト」に変更
2. ブラウザをリロード（F5）
3. → 「システム設定」タブで「デフォルト」が選択されたままを確認
4. **テスト後は「鍼灸院」に戻すこと**

---

## テスト3: 音声入力でのプリセット効果（オプション）

### 準備
- 顧客詳細ページで音声入力機能を使用

### テスト内容
1. **鍼灸院プリセット**で音声入力:
   「今日は腰が痛いって言ってて、じんゆとだいちょうゆに鍼をうちました」
   → 「腎兪」「大腸兪」と正しく変換されるか確認

2. 専門用語テスト:
   「がいせん制限があって内線も痛い」
   → 「外旋」「内旋」と変換されるか確認
